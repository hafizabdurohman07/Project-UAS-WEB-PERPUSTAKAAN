# Library Application (MVC & OOP)

Aplikasi Manajemen Perpustakaan sederhana yang dibangun dengan prinsip **OOP (Object Oriented Programming)** dan arsitektur **MVC (Model-View-Controller)**.

## Fitur Utama
- **Routing Custom**: Menggunakan `.htaccess` untuk URL yang bersih.
- **Responsive UI**: Dibangun dengan **Bootstrap 5** (Mobile First).
- **Multi-Role Login**: Akses berbeda untuk Admin dan User/Member.
- **CRUD Lengkap**: Pengelolaan data Buku dan Anggota.
- **Fitur Lanjutan**: 
    - **Filter Pencarian**: Cari data secara instan.
    - **Pagination**: Pembagian halaman untuk efisiensi tampilan.

## Struktur Directory
- `app/`: Berisi inti aplikasi (Core, Controllers, Models, Views).
- `public/`: Assets publik dan entry point `index.php`.

## Instalasi
1. Clone repository atau ekstrak file ke folder `xampp/htdocs/perpustakaan`.
2. Pastikan database `perpustakaan` sudah diimport (Gunakan file SQL yang disediakan).
3. Akses `http://localhost/perpustakaan` di browser.
